### 这是啥？

PageCMS 是面向静态站点、静态博客设计的 SaaS 可视化内容管理平台，核心用于托管、管理博客文档、图文内容等静态资源，适配基于 GitHub 源码托管的静态博客

### 原理

PageCMS 依靠静态项目根目录下的 `.page.yml` 配置文件，以声明式语法完成全部功能定义，例如以下配置项：

1. 博文 Markdown 文件存储目录（如 `source/_posts`）；
2. 博客图片、静态素材存放的资源目录；
3. 绑定仓库的 GitHub Actions CI/CD 工作流；
4. 在线页面内触发站点重新构建的交互按钮行为。

完成声明后，平台会通过 GitHub 官方 API 读写仓库文件、触发构建流程，替代本地传统发文流程，我们不再需要手动本地启动本地的 VS Code、打开项目源码文件夹，手动新建博文、创建分类目录、导入配图等可脚本化的重复操作，这一切全部可在在线平台[https://app.pagescms.org/](https://app.pagescms.org/) 里全部自动化完成
